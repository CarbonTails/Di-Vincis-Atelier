Review the current staged changes (or the changes described by the user) for:

- Bugs and logic errors
- Security issues (injection, secrets, unsafe input handling)
- Style and convention violations
- Missing tests or documentation

Report findings grouped by severity (Critical, High, Medium, Low) and suggest concrete fixes for each.
